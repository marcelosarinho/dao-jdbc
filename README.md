# DAO-JDBC

Projeto desenvolvido para aprender **JDBC (Java Database Connectivity)** utilizando o padrão de projeto **DAO (Data Access Object)** em Java.

O objetivo principal foi entender como funciona a conexão com banco de dados relacional, execução de comandos SQL e organização do código separando a lógica de acesso a dados da regra de negócio.

---

## 🧠 Tecnologias Utilizadas

- Java
- JDBC
- MySQL
- Padrão DAO

---

## 🎯 Objetivos do Projeto

- Aprender a configurar conexão com banco usando JDBC
- Executar operações CRUD (Create, Read, Update, Delete)
- Implementar o padrão DAO para organizar o código
- Separar entidades, regras e acesso a dados
- Trabalhar com `PreparedStatement`, `ResultSet` e tratamento de exceções
